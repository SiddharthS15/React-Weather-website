About SkyWeather

🌤️ Weather API

This app uses the OpenWeatherMap API to fetch real-time weather data from around the world.

  • Temperature in Celsius
  • Humidity percentage
  • Wind speed in m/s
  • Rainfall data (when available)
  • Weather conditions and icons

🔍 How to Search

Simply enter any city name in the search bar:

  • City only: "London", "Tokyo", "Paris"
  • City with country: "New York, US"
  • Major cities: Works with most global cities
  • Press Enter: Or click the search button

🌙 Theme Toggle

Switch between light and dark modes using the toggle in the navigation:

  • ☀️ Light Mode: Default bright theme
  • 🌙 Dark Mode: Easy on the eyes
  • Automatic Save: Your preference is remembered
  • Smooth Transition: All elements adapt instantly

✨ Features

  • Real-time weather data
  • Beautiful weather icons
  • Loading animations

  • Responsive design
  • Keyboard shortcuts (Enter to search)
